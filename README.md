# jk-map

A map of local businesses in a Japanese town.

## Demo
https://code4fukui.github.io/jk-map/

## Features
- Interactive map showing local businesses
- Information on business name, phone, category, description, and more
- Provided as an HTML page with a CSV data file

## Usage
The project is a static website hosted on GitHub Pages. To use, simply open the `index.html` file in a web browser.

## Data
The business information is stored in a CSV file (`jk-map.csv`) which is loaded and displayed on the map.
