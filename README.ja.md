# jk-map

「JKまっぷ」は、福井県の観光スポットをマッピングしたウェブアプリです。

## デモ
https://code4fukui.github.io/jk-map/

## 機能
- 観光スポットの一覧と詳細情報の表示
- 地図上でのスポットの位置確認

## 使い方
ウェブブラウザで上記のデモページにアクセスすると、JKまっぷを使えます。

## データ
本プロジェクトでは、CSV形式のデータファイル `jk-map.csv` を利用しています。このファイルには、観光スポットの名称、電話番号、カテゴリー、アイコン、概要、定休日、緯度経度などの情報が含まれています。
